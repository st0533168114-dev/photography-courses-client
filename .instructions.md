# client – מדריך פרויקט

## טכנולוגיות
React 19, Vite 8, Redux Toolkit, React Router v7, Axios, CSS Modules, Swiper, ESLint. JavaScript בלבד (`.jsx`/`.js`).

## מבנה קבצים
```
src/
├── App.jsx, main.jsx, index.css
├── pages/ [HomePage, AboutPage, CoursesPage, CategoriesPage, CourseDetailsPage, 
│           ShoppingCartPage, CreateOrderPage, PaymentConfirmationPage, 
│           LoginPage, SignUpPage, ContactUsPage, MyCoursesPage, admin/]
├── components/ [CourseCard, CategoryItem, AddToCartButton, DeleteFromCartButton,
│               CategoryGrid, CommentsSection, CommentItem, FAQSection, FAQItem,
│               WellcomeSection, ContactUsSection, PurchaseButton, MyCourseItem,
│               CourseCartItem]
├── layout/ [Header.jsx, Footer.jsx]
├── API/ [axiosConfig.js, coursesApi.js, categoryApi.js, userApi.js, 
│         orderApi.js, paymentApi.js, shoppingCartApi.js, faqApi.js]
├── redux/ [store.js, slices/]
├── CSS/ [pages/, components/, layout/ → *.module.css]
├── styles/ [variables.css]
├── assets/, classes/
└── public/
```

## ארכיטקטורה
- SPA: `BrowserRouter` + Routes ב-`App.jsx`; `checkAuth` בטעינה
- זרימה: API → `createAsyncThunk` בסלייס → דפים/רכיבים עם `useSelector`/`useDispatch`
- Axios משותף; Bearer token מ-`localStorage` דרך interceptor
- שרת: `http://localhost:1234`; מזהי MongoDB: `_id`

## סגנון קוד ושמות
- רכיבים: `export default function Xxx()`; דפים עוטפים Header + `<main>` + Footer
- CSS Modules: `import styles from "../CSS/..."`
- Thunks: `"domain/action"`; סלייסים: `initialState`, `reducers`, `extraReducers`
- שמות: PascalCase לרכיבים/דפים, camelCase+`Api`/`Slice` לקבצי לוגיקה
- ממשק והערות בקוד בעברית; שמור על מוסכמות קיימות (למשל `WellcomeSection`, `/ShoppingCartPage`)

---
**הערה:** מבנה הקבצים למעלה יעודכן בכל פעם שנוספים או מתווספים קבצים חדשים לפרויקט.
